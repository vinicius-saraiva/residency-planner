## Usage

### Viewing Analysis Templates
Each file is a markdown document containing the structured template. Open any file to see the complete analysis framework.

### Completing an Analysis
1. Open the analysis file for your target competitor
2. Research each section using:
   - Company investor relations pages
   - Annual reports (10-K filings)
   - Financial news and analyst reports
   - Industry research reports
   - Customer review sites
   - Social media presence
3. Fill in each section with factual, data-driven information
4. Include sources and references at the bottom

### AI-Powered Analysis
To automatically generate research-backed analysis:

```bash
# Set your API key
export ANTHROPIC_API_KEY="sk-ant-..."

# Generate full analysis (will overwrite template)
npm run analyze-ai "JPMorgan Chase"
```

### Batch Processing
To regenerate all templates:

```bash
./scripts/batch-analyze.sh
```
